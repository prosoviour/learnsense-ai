# **LearnSense AI \- Version 1.0**

## **Project Overview**

LearnSense AI is an emotion-aware tutoring system designed to adapt learning experiences based on student emotions and engagement levels.

## **Version 1 Features Completed**

### **Authentication & Entry**

* Login screen with Name, Email, and Learning Topic inputs  
* Start Learning workflow

### **Student Dashboard**

* Personalized welcome screen  
* Topic display  
* Progress tracking placeholder  
* Session count placeholder  
* Emotion tracking status

### **AI Tutor Interface**

* Chat interface  
* Student message bubbles  
* AI Tutor message bubbles  
* Send message functionality  
* Navigation back to dashboard

### **Emotion Intelligence Panel**

* Current Emotion selector  
* Confidence score  
* Tutor Mode  
* Attention Score  
* Learning State

### **Dynamic Emotion Adaptation**

* Engaged → Advanced Learning  
* Confused → Guided Learning  
* Bored → Interactive Learning  
* Frustrated → Supportive Coaching

### **Analytics**

* Emotion History tracking  
* Session Analytics panel  
* Current Emotion summary  
* Confidence summary
