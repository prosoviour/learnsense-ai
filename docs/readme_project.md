**LearnSense AI**

LearnSense AI is an emotion-aware intelligent tutoring prototype that adapts tutor responses based on learner emotions.

 **Features**

* Login screen  
* Student dashboard  
* AI tutor chat screen  
* Emotion selector  
* Dynamic confidence score  
* Dynamic attention score  
* Dynamic tutor mode  
* Emotion history  
* Session analytics

 **Tech Stack**

* React  
* Vite  
* JavaScript  
* CSS

 Status  
Version 1 frontend prototype completed.
